# Obtendo o atributo

Escreva o código para selecionar o elemento do documento com o atributo `data-widget-name` e leia seu valor.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Escolha o gênero</div>

  <script>
    /* seu código */
  </script>
</body>
</html>
```
