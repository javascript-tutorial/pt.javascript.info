# Reescreva 'if..else' usando '?'

Reescreva `if..else` usando vários operadores ternários `'?'`.

Para facilitar a leitura, é recomendável dividir o código em várias linhas.

```js
let message;

if (login == 'Funcionário') {
  message = 'Olá';
} else if (login == 'Diretor') {
  message = 'Saudações';
} else if (login == '') {
  message = 'Não logado';
} else {
  message = '';
}
```
