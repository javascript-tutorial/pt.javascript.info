# Mostre o sinal

Usando `if..else`, escreva o código que obtém um número via `prompt` e depois mostra em `alert`:

- `1`, se o valor for maior que zero,
- `-1`, se menor que zero,
- `0`, se igual a zero.

Nesta tarefa assumimos que a entrada é sempre um número.
