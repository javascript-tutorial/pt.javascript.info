# Exiba números primos

Um número inteiro maior que `1` é chamado de [primo](https://pt.wikipedia.org/wiki/N%C3%BAmero_primo) se não puder ser dividido sem resto por nada exceto `1` e ele mesmo.

Em outras palavras, `n > 1` é primo se não puder ser dividido igualmente por nada exceto `1` e `n`.

Por exemplo, `5` é primo, porque não pode ser dividido sem resto por `2`, `3` e `4`.

**Escreva o código que exibe números primos no intervalo de `2` a `n`.**

Para `n = 10` o resultado será `2,3,5,7`.

P.S. O código deve funcionar para qualquer `n`, não ser ajustado para qualquer valor fixo.
