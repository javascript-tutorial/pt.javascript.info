# Quais valores o laço "for" mostra?

Para cada laço, escreva quais valores ele vai mostrar. Depois compare com a resposta.

Ambos os laços exibem os mesmos valores via `alert`, ou não?

1. A forma pós-fixa:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. A forma prefixa:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
