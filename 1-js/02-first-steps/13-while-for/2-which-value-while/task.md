# Quais valores o laço while mostra?

Para cada iteração do laço, escreva qual valor ele exibe e depois compare com a solução.

Ambos os laços exibem os mesmos valores via `alert`, ou não?

1. A forma prefixa `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. A forma pós-fixa `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
