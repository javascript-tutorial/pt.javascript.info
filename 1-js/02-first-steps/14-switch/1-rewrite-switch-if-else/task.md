# Transforme o "switch" num "if"

Escreva o código empregando `if..else` que corresponda ao seguinte `switch`:

```js
switch (browser) {
  case 'Edge':
    alert( "Você tem o Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'Ok, nós também suportamos esses navegadores' );
    break;

  default:
    alert( 'Esperamos que esta página pareça bem!' );
}
```
